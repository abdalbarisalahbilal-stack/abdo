# Store Manager

برنامج **Python** لإدارة المتاجر بطريقة بسيطة وفعّالة.

---

## 📌 الوصف

هذا المشروع يسمح لك بـ:

- حفظ أسماء المنتجات في المتجر.
- إضافة تفاصيل المتجر (المدينة، وقت العمل، عدد الموظفين...).
- تخزين كل البيانات في ملف `store.txt`.
- عرض محتوى الملف بسهولة.
- يدعم أي عدد من المنتجات وأي تفاصيل إضافية.

---

## ⚡ مميزات المشروع

- استخدام **المعاملات المتغيرة** `*products` و `**details` لتخزين أي عدد من المنتجات والتفاصيل.
- يمكن تشغيل البرنامج مباشرة أو استدعاء الدالة في مشاريع أخرى.
- حفظ كل سجل في الملف مع فاصل بين المتاجر لتسهيل القراءة.
- سهل الاستخدام حتى للمبتدئين.

---

## 🛠️ كيفية الاستخدام

1. استدعاء الدالة `store()`:

```python
from store import store

store(
    "Abdo",                   # اسم صاحب المتجر
    "Laptop",                  # المنتجات
    "Phone",
    "Headset",
    city="Kuala Lumpur",       # تفاصيل إضافية
    open_time="8 - AM",
    worker=12
)
